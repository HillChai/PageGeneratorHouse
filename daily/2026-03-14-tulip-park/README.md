# Tulip Sea, People Mountain

A romantic, single-page tribute to a crowded tulip park. The hero photo is softened with a dusk overlay, petals drift on tap, and visitors can save a little memory in the page.

**Interactions**
- Tap “Scatter petals” to release a shower of floating petals.
- Tap lines in the poem to make them glow.
- Save a short memory in the “Memory jar”.

**Assets**
- `tulips.jpg` (user-provided photo)
