# Afterglow Echoes

A single-page mood piece about a late-night conversation that fades by morning.

## Interactions
- Click any message bubble to let it fade into an echo.
- Use the **Distance** slider to push the chat further away (more blur/less opacity).
- Tap **Hold the last message** to keep one bubble clear.
- Type a message and send it; it appears briefly, then drifts away.
