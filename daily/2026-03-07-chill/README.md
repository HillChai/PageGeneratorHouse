# 2026-03-07 — Slow Morning

Chill weekend one-pager with a warm sunlight canvas. Users can steer the light, tap to breathe in a puff of steam, toggle dim, and dial warmth/glow/breeze.

Interactions:
- Move cursor to guide sunlight
- Tap/click to add a warm puff
- Brew tea button to spawn steam
- Dim room toggle
- Sliders for warmth/glow/breeze
