# 2026-03-07 — Slow Morning

Chill weekend one-pager with a warm sunlight canvas. Users can steer the light, tap to breathe in a puff of steam, and dial warmth/glow/breeze.

New interaction:
- Taps build "morning energy". Faster taps brighten the room more.
- After 12 taps the room stays bright (won't fully darken).
- Initial state is a dim morning; taps gradually open the room.

Interactions:
- Move cursor to guide sunlight
- Tap/click to add a warm puff + brighten
- Brew tea button to spawn steam
- Dim room toggle
- Sliders for warmth/glow/breeze
