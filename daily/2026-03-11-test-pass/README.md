# Quiet Triumph

A single-page celebration of passing a test after a busy, nervous night. The page moves from tension to relief with a momentum slider, floating "relief notes," and a confetti release.

## Interactions
- **Release the confetti** button for a celebratory burst
- **Breathe with me** toggle to soften the mood
- **Momentum Meter** slider to replay the climb to passing
- **Relief Notes** buttons to float worries into the sky
- **Glow Log** to add a personal win
