# Bicycle Flow

A playful, fluent one-page ride inspired by daily bike commutes. Hold the pedal button or press Space to build speed, watch the skyline glide, and ring the bell for extra joy.

**Interactions**
- Hold **Pedal** (or Space) to accelerate; release to coast.
- Toggle **Playful mode** for gentle bike wobble.
- Click **Ring Bell** for a quick chime.
